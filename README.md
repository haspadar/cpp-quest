# C++ Quest

Курс по основам объектно-ориентированного программирования в C++.

Строим RPG-движок шаг за шагом: персонажи, инвентарь, способности, монстры, бой.
Каждый урок — теория и задачи на одну тему ООП.

Курс рассчитан на тех, кто уже знаком с основами C++:
- переменные, типы данных, приведение типов
- условия (`if`, `switch`) и циклы (`for`, `while`)
- функции, передача аргументов по значению и по ссылке
- массивы, строки (`std::string`)
- указатели и ссылки
- структуры (`struct`)

## Как скачать

```bash
git clone https://github.com/haspadar/cpp-quest.git
cd cpp-quest
```

## Компиляция примеров

```bash
g++ -std=c++17 -Wall -Wextra -o main main.cpp
./main
```
