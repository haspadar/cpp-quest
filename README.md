# C++ Quest

Курс по объектно-ориентированному программированию в C++.

Учим правильному ООП: маленькие объекты с поведением, композиция вместо наследования, декораторы, интерфейсы, стратегии. Всё через практику — строим RPG-движок шаг за шагом.

Между уроками — привалы: паузы на лабораторные работы (БНТУ, ОАП), где применяем изученное к заданиям из универа.

Курс рассчитан на тех, кто уже знаком с основами C++:
- переменные, типы данных, приведение типов
- условия (`if`, `switch`) и циклы (`for`, `while`)
- функции, передача аргументов по значению и по ссылке
- массивы, строки (`std::string`)
- указатели и ссылки
- структуры (`struct`)

## Как скачать

```bash
git clone https://github.com/haspadar/cpp-quest.git
cd cpp-quest
```

## Компиляция примеров

```bash
g++ -std=c++17 -Wall -Wextra -o main main.cpp
./main
```
