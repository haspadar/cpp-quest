# 01. Переход с C на C++

В C ты работал с `printf`, `scanf`, `char*`, `strcmp`. В C++ для всего этого есть более удобные инструменты. В этом уроке — знакомство с ними.

## Подключение библиотек: #include

В C ты уже использовал `#include` — например, `#include <stdio.h>` для `printf`. В C++ всё так же: чтобы использовать что-то из стандартной библиотеки, нужно подключить нужный заголовочный файл.

Разница — в C++ заголовки пишутся без `.h`:

```cpp
// C
#include <stdio.h>
#include <string.h>

// C++
#include <iostream>   // cout, cin
#include <string>     // std::string
```

Каждый `#include` даёт доступ к определённому набору инструментов. Нужен вывод на экран — подключай `<iostream>`. Нужны строки — подключай `<string>`.

### std:: — что это за префикс

Всё из стандартной библиотеки C++ живёт в пространстве имён `std`. Поэтому пишем `std::cout`, `std::string`, `std::cin`.

Это как фамилия: в школе может быть два Саши, но `Иванов::Саша` и `Петров::Саша` — разные люди. `std::` говорит: "это `cout` из стандартной библиотеки, а не что-то другое".

## Строки: std::string вместо char*

В C строки — это массивы символов:

```cpp
char name[] = "Archer";
char* ptr = "Warrior";
```

Это работает, но неудобно: нужно следить за размером массива, выделять память, не забывать про `\0`. Сравнить две строки через `==` нельзя — нужен `strcmp`.

В C++ есть тип `std::string` — строка, которая сама управляет памятью:

```cpp
#include <string>

std::string name = "Archer";
```

Что умеет `std::string`, чего не умеет `char*`:

```cpp
std::string a = "Hello";
std::string b = "World";

std::string c = a + " " + b;  // склейка через +
bool same = (a == b);         // сравнение через ==
int len = a.length();         // длина без strlen
```

Не нужно думать про размер буфера, `malloc`, `\0`. Строка растёт сама.

## Вывод: cout вместо printf

```cpp
#include <iostream>

int main() {
    int hp = 100;
    std::cout << "Health: " << hp << "\n";
    return 0;
}
```

`<<` — оператор вывода. Можно склеивать несколько значений в цепочку. Тип определяется автоматически — не нужен `%d`, `%s` и прочие спецификаторы.

Сравни:

```cpp
// C
printf("Name: %s, HP: %d\n", name, hp);

// C++
std::cout << "Name: " << name << ", HP: " << hp << "\n";
```

## Ввод: cin вместо scanf

```cpp
int damage;
std::cout << "Damage: ";
std::cin >> damage;
```

`>>` — оператор ввода. Тоже определяет тип сам — не нужен `%d`.

Можно читать несколько значений подряд:

```cpp
int hp, armor;
std::cin >> hp >> armor;
```

Для строк с пробелами `cin >>` не подходит — он читает до первого пробела. Если нужна целая строка:

```cpp
std::string name;
std::getline(std::cin, name);
```

## Русский язык в консоли Windows

По умолчанию консоль Windows использует кодировку CP866, а Visual Studio сохраняет файлы в CP1251. Из-за этого русские буквы превращаются в кракозябры.

Чтобы русский текст нормально выводился и вводился, добавь две строки в начало `main`:

```cpp
#include <iostream>
#include <Windows.h>

int main() {
    SetConsoleCP(1251);       // кодировка ввода
    SetConsoleOutputCP(1251); // кодировка вывода

    std::cout << "Привет, мир!\n";
    return 0;
}
```

`SetConsoleCP` — задаёт кодировку для ввода с клавиатуры.
`SetConsoleOutputCP` — задаёт кодировку для вывода на экран.
1251 — это Windows-1251, стандартная кириллическая кодировка Windows.

Без этих строк `cout << "Привет"` выведет мусор.

## Шаблон программы

Каждая программа в этом курсе начинается так:

```cpp
#include <iostream>
#include <string>
#include <Windows.h>

int main() {
    SetConsoleCP(1251);
    SetConsoleOutputCP(1251);

    // код здесь

    return 0;
}
```

## Шпаргалка

| Задача | C | C++ |
|---|---|---|
| Подключение вывода | `#include <stdio.h>` | `#include <iostream>` |
| Вывод на экран | `printf("HP: %d\n", hp)` | `std::cout << "HP: " << hp << "\n"` |
| Ввод числа | `scanf("%d", &x)` | `std::cin >> x` |
| Объявление строки | `char name[] = "Archer"` | `std::string name = "Archer"` |
| Склейка строк | `strcat(a, b)` | `a + b` |
| Сравнение строк | `strcmp(a, b) == 0` | `a == b` |
| Длина строки | `strlen(s)` | `s.length()` |
| Русский язык (Windows) | — | `SetConsoleCP(1251); SetConsoleOutputCP(1251);` |

## Практика

### 1.1. Визитка персонажа

Программа спрашивает имя и класс персонажа, потом выводит визитку.

Пример работы:

```text
Имя: Артём
Класс: Лучник
----
Артём the Лучник
Длина имени: 5
```

Используй `std::getline` для ввода (имя может содержать пробелы), `+` для склейки, `.length()` для длины.

### 1.2. Калькулятор урона

Программа запрашивает силу атаки и броню цели. Считает и выводит фактический урон (атака минус броня, но не меньше нуля).

Пример:

```text
Сила атаки: 40
Броня цели: 25
Фактический урон: 15
```

Ещё пример:

```text
Сила атаки: 10
Броня цели: 30
Фактический урон: 0
```

### 1.3. Сравнение паролей

Программа запрашивает у игрока пароль для входа в гильдию. Правильный пароль — `"dragon"`. Если совпадает — `"Добро пожаловать!"`, если нет — `"Доступ запрещён"`.

Пример:

```text
Пароль: dragon
Добро пожаловать!
```

```text
Пароль: knight
Доступ запрещён
```

Используй `std::string` и сравнение через `==`.
