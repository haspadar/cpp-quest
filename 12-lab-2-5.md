## Потоки C++ — это объекты

Ты знаешь Unix: `echo "hello" > file.txt` — перенаправление stdout в файл. В C++:

- `std::cout` — stdout (экран)
- `std::cin` — stdin (клавиатура)
- `std::cerr` — stderr (ошибки)
- `std::ofstream` — stdout в файл
- `std::ifstream` — stdin из файла

Все они — объекты с `<<` и `>>`. Ты уже перегружал `operator<<` в уроке 05. Тот же оператор работает и с файлами:

```cpp
#include <fstream>

std::ofstream file("report.txt");
file << knight << "\n";   // то же самое, что cout <<
file.close();
```

## Манипуляторы: форматирование вывода

Для табличного вывода нужны манипуляторы из `<iomanip>`:

```cpp
#include <iomanip>

// setw(n) — ширина поля
std::cout << std::setw(15) << "Name"
          << std::setw(10) << "HP"
          << std::setw(10) << "Damage" << "\n";

// left — выравнивание по левому краю
std::cout << std::left
          << std::setw(15) << "Knight"
          << std::setw(10) << 100
          << std::setw(10) << 25 << "\n";
```

```text
           Name        HP    Damage
Knight         100       25
```

Основные манипуляторы:

| Манипулятор | Что делает |
|---|---|
| `setw(n)` | Ширина следующего поля (только одного!) |
| `left` | Выравнивание по левому краю |
| `right` | Выравнивание по правому краю (по умолчанию) |
| `setprecision(n)` | Количество знаков после точки (для `double`) |
| `fixed` | Фиксированная точка (не научная нотация) |
| `setfill(c)` | Символ-заполнитель (вместо пробела) |
| `hex` / `oct` / `dec` | Система счисления |

Важно: `setw` действует только на **следующее** значение. Остальные — до отмены.

## Таблица объектов

```cpp
#include <iostream>
#include <iomanip>
#include <string>

void printTable(Player players[], int count) {
    // Заголовок
    std::cout << std::left
              << std::setw(15) << "Имя"
              << std::setw(8)  << "HP"
              << std::setw(15) << "Оружие"
              << std::setw(8)  << "Урон"
              << "\n";

    std::cout << std::setfill('-') << std::setw(46) << "" << "\n";
    std::cout << std::setfill(' '); // вернуть пробел

    // Данные
    for (int i = 0; i < count; i++) {
        std::cout << std::left
                  << std::setw(15) << players[i].getName()
                  << std::setw(8)  << players[i].getHp()
                  << std::setw(15) << players[i].getWeaponName()
                  << std::setw(8)  << players[i].getDamage()
                  << "\n";
    }
}
```

```text
Имя            HP      Оружие         Урон
----------------------------------------------
Knight         100     Sword          25
Mage           60      Staff          40
Archer         80      Bow            22
```

## Запись в файл

```cpp
#include <fstream>

void saveReport(Player players[], int count, const std::string& filename) {
    std::ofstream file(filename);

    if (!file.is_open()) {
        std::cerr << "Не удалось открыть файл: " << filename << "\n";
        return;
    }

    file << "=== Ведомость отряда ===\n\n";

    file << std::left
         << std::setw(15) << "Имя"
         << std::setw(8)  << "HP"
         << std::setw(15) << "Оружие"
         << "\n";

    for (int i = 0; i < count; i++) {
        file << std::left
             << std::setw(15) << players[i].getName()
             << std::setw(8)  << players[i].getHp()
             << std::setw(15) << players[i].getWeaponName()
             << "\n";
    }

    file.close();
    std::cout << "Ведомость сохранена в " << filename << "\n";
}
```

`ofstream` и `cout` — одинаковый синтаксис. Если объект умеет `operator<<` — он умеет записываться и на экран, и в файл.

## Чтение из файла

```cpp
#include <fstream>

std::ifstream file("data.txt");
std::string name;
int hp;

while (file >> name >> hp) {
    std::cout << "Загружен: " << name << ", " << hp << " HP\n";
}

file.close();
```

## Сравнительный анализ (задание 2 лабы)

| | C (`printf/scanf`) | C++ (потоки) |
|---|---|---|
| Типобезопасность | Нет — `%d` для int, `%s` для строки, перепутал — краш | Да — `cout << x` сам определяет тип |
| Расширяемость | Нельзя научить выводить свой класс | `operator<<` для любого типа |
| Форматирование | `printf("%-15s %5d", name, hp)` — компактно, но криптично | `setw(15) << left` — длиннее, но понятнее |
| Ввод строк | `scanf("%s")` не читает пробелы, `gets` небезопасен | `getline(cin, str)` — безопасно |
| Файлы | `FILE*, fprintf, fscanf` | `ofstream, ifstream` — те же `<<` и `>>` |

## Напоследок

1. `cout` и `cin` — это объекты классов `ostream` и `istream`. Когда ты пишешь `cout << 42`, вызывается метод `ostream::operator<<(int)`. Это ООП, встроенное в стандартную библиотеку.

2. В Unix всё — файл. В C++ всё — поток. `cout`, файл, строковый поток (`stringstream`) — один и тот же интерфейс `<<` / `>>`. Если знаешь один — знаешь все.

3. Манипулятор `endl` не просто переводит строку — он ещё сбрасывает буфер (`flush`). Для обычного вывода `"\n"` быстрее. `endl` нужен, когда важно, чтобы текст появился на экране немедленно (например, перед `cin`).
